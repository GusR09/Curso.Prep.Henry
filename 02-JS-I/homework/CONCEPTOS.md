CONCEPTOS (Explicados para niños de 12 años):

VARIABLES:         Es un elemento que se emplea para almacenar y hacer referencia a otro valor.
STRINGS:           Es un texto doble o simple entre comillas ("strings" o "estrings dobles").
FUNCIONES:         Es un conjunto de isntrucciones que realiza una tarea o calcula algun valor tiene que toamr una 
			       entrada y devolver una salida donde haya relaciom obvia entre llas.
DENOMINACIONES:    Son las intrucciones que se les da para cumplir una funcion, lo mas recomendado es cerrarlas    
                   con punto y coma (;).
VALORES BOOLEANOS: Se usan para almacenar información del tipo verdadero/falso, cumple/no cumple, si/no
                   (informacion que solo tenga 2 estados).
